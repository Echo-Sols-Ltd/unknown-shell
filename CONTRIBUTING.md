# Contributing to Unknown Shell

Thank you for your interest in contributing to Unknown Shell! This document provides guidelines and instructions for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/unknown-shell.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -m 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature-name`
7. Open a Pull Request

## Code Style

- Use TypeScript for all new code
- Follow the existing code style
- Use ESLint and Prettier
- Write meaningful commit messages

## Testing

Before submitting a PR, make sure:
- All tests pass
- The code builds successfully
- No linting errors

## Questions?

Feel free to open an issue for any questions or concerns.
